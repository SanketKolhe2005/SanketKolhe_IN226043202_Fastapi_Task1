# Data Science Internship – Logic Building Task 1

## Overview

This repository contains solutions for **Logic Building Task – 1** assigned during the **Data Science Internship – February 2026**.
The tasks focus on developing basic **Python programming and problem-solving skills** used in real-world data science applications.

The solutions are written using **clean, simple, and well-commented Python code**.

---

# Tasks Implemented

## 1. User Login Check

This program checks whether a user login is successful by comparing the entered username and password with the correct credentials.

**Concepts Used**

* Conditional statements (if–else)
* Basic authentication logic

**Real-World Application**
Used in login systems and authentication mechanisms.

---

## 2. Pass / Fail Analyzer

This program analyzes a list of student marks and determines how many students **passed** and **failed**.

**Condition**

* Pass: Marks ≥ 50
* Fail: Marks < 50

**Concepts Used**

* Lists
* Loops
* Conditional statements

**Real-World Application**
Student result processing systems.

---

## 3. Simple Data Cleaner

This program cleans and standardizes a list of names by:

* Removing extra spaces
* Converting names to lowercase

**Concepts Used**

* String manipulation
* strip() and lower() functions

**Real-World Application**
Data preprocessing before analysis in data science.

---

## 4. Message Length Analyzer

This program analyzes messages and:

* Prints the length of each message
* Flags messages longer than 10 characters.

**Concepts Used**

* Loops
* String length function (len)

**Real-World Application**
Text filtering and message validation systems.

---

## 5. Error Message Detector

This program analyzes system logs and counts how many times the **ERROR** message appears.

**Concepts Used**

* Loops
* Counting occurrences

**Real-World Application**
System monitoring and log analysis.
